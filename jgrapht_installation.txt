1. Download jgrapht-1.1.0.zip file from here: http://jgrapht.org/

2. Unzip file NOT into the github directory

3. In Eclipse open:
	Window->Preferences->java->User Libraries
	
4. In User Libraries select "New"

4a. Name the library jgrapht
4b. Select "Add External JARs" and select the lib directory inside the jgrapht folder previously unzipped
4c. Add those JAR files to the library. 

5. Select Build Path by right clicking on the project in eclipse and choose "Add Library"

6. Select User Library and import jgrapht to the project.
